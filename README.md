# GPTPulse

GPTPulse is a small Chrome extension for ChatGPT.

It helps keep long chats easier to read by showing only the most recent messages. Older messages are not deleted. They are only hidden locally in your browser.

The idea is simple: large chat histories can start to feel heavy over time. GPTPulse keeps the view lighter and cleaner.

## What GPTPulse does

GPTPulse shows only the last **N messages** in ChatGPT.

You can choose how many messages stay visible. The default value is **10**.

There is also a small overlay in the top right corner of the chat. It gives you a quick overview of:

Messages  
Hidden  
Chars  
Visible messages

You can change the value directly with the slider.

## What GPTPulse does not do

GPTPulse does not delete messages.  
GPTPulse does not change anything in your account.  
GPTPulse does not send data anywhere.  
GPTPulse works only locally in your browser.

## Why it is useful

When a chat gets very long, the page can start to feel cluttered and slow.  
With GPTPulse, only the most relevant recent part stays visible.  
That makes long conversations feel cleaner and easier to manage.

## Installation

1. Open `chrome://extensions`
2. Enable Developer mode
3. Click **Load unpacked**
4. Select the `GPTPulse` folder
5. Reload ChatGPT

## Usage

Once the extension is active, a small overlay appears in the top right corner of the chat.

There you can quickly see:

how many messages are currently visible  
how many are hidden  
roughly how large the chat is  
which visible limit is active

Use the slider to control how many messages stay visible.

## Settings

In the extension settings, you can:

set the visible message limit  
show or hide the overlay

Changes are saved automatically.

## Privacy

GPTPulse works completely locally.

There is no login  
no server  
no tracking  
no upload of your chats

Everything happens only in your browser.

## Current status

GPTPulse is intentionally small and focused.

The current goal is:

a clean display  
a compact overlay  
simple controls  
very little overhead

## Note

ChatGPT can change its interface at any time.  
If OpenAI updates the page layout, GPTPulse may need adjustments.
